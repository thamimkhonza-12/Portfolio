# Thamsanqa Dlamini Full-Stack Developer Portfolio

Static personal portfolio built with HTML, CSS, and JavaScript.

## Use locally

Open `index.html` in a browser.

## Deploy

Upload the folder to GitHub Pages, Netlify, Vercel, or any static hosting service.

## Files

- `index.html` - page content and sections
- `style.css` - responsive visual design
- `script.js` - mobile navigation and smooth scrolling
- `resume/Thamsanqa_Dlamini_CV.pdf` - downloadable CV
